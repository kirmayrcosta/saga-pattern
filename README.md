


## Description

Exemplo do projeto com Saga Pattern

Baixar o projeto Zeebe https://github.com/zeebe-io/zeebe-docker-compose

acessar a pasta operate

```
docker-compose up -d
```

Subir o projeto Order_servico.bpmn pelo Zeebe Modeler

Dar NPM install no projeto e depois npm start:dev


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
